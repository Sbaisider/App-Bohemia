# API Reference

## Autenticazione
- `POST /api/login`
- `GET /api/me`

## Magazzino
- `GET /api/magazzino`
- `POST /api/magazzino/carico`
- `POST /api/magazzino/scarico`

## Cambusa
- `GET /api/cambusa`
- `POST /api/cambusa/carico`
- `POST /api/cambusa/scarico`

## Tavoli
- `GET /api/tavoli`
- `POST /api/tavoli/prenotazione`
- `POST /api/tavoli/checkin`
