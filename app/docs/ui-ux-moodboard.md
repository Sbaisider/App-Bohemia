# UI/UX Moodboard

## Palette Colori
- Nero, viola scuro, accenti neon (verde, blu, rosa)

## Font Consigliati
- Montserrat
- Space Grotesk

## Elementi UI
- Navbar fissa
- Card evento con glow effect
- Pulsanti arrotondati e animati

## UX Notes
- Design mobile-first
- Accesso rapido a sezioni principali
- Feedback visivo immediato per azioni
